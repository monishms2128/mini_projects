# 🎮 Tic Tac Toe AI

A smart Tic Tac Toe AI game developed using Python and Tkinter GUI as part of the CODSOFT Artificial Intelligence Internship.

## 🚀 Features

- Player vs Computer gameplay
- Smart AI opponent
- GUI interface using Tkinter
- Live scoreboard tracking
- Restart game functionality
- Winner and draw popup messages
- Modern dark theme UI

## 🛠️ Technologies Used

- Python
- Tkinter
- Random Module

## ▶️ How to Run

1. Open terminal in project folder
2. Run the command:

```bash
python gui_tic_tac_toe.py
```
