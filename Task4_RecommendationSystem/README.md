# 🎬 Movie Recommendation System

A GUI-based Movie Recommendation System developed using Python and Tkinter as part of the CODSOFT Artificial Intelligence Internship.

## 🚀 Features

- Movie recommendation engine
- Content-based filtering
- GUI application using Tkinter
- Random movie suggestions
- Dark modern UI
- Interactive recommendation results
- Error handling popup messages
- Enter key support

## 🛠️ Technologies Used

- Python
- Tkinter
- Pandas
- Random Module

## ▶️ How to Run

1. Open terminal in project folder
2. Run the command:

```bash
python gui_recommendation_system.py
```

## 📸 Project Demo

The system can:
- Recommend movies based on genre similarity
- Display recommendations in GUI
- Handle invalid movie inputs
- Provide interactive user experience
