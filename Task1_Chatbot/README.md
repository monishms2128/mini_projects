# 🤖 MONISH AI CHATBOT

A smart rule-based AI chatbot developed using Python and Tkinter GUI as part of the CODSOFT Artificial Intelligence Internship.

## 🚀 Features

- Rule-based chatbot responses
- GUI chatbot interface
- Random greeting responses
- Time and date functionality
- Joke generation
- Motivation responses
- AI-related answers
- Enter key support
- Modern dark theme UI

## 🛠️ Technologies Used

- Python
- Tkinter
- Random Module
- Datetime Module

## ▶️ How to Run

1. Open terminal in project folder
2. Run the command:

```bash
python gui_chatbot.py
```
